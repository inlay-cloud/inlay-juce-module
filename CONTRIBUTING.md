# Contributing

Thanks for your interest in improving the Inlay JUCE module.

This repository is public-source proprietary software, not an open source
project. The module license is in
[`modules/inlay_product_unlocking/LICENSE`](modules/inlay_product_unlocking/LICENSE).

## Issues

Use GitHub issues for bug reports, integration problems, and documentation
feedback. For security-sensitive reports, use the process in
[`SECURITY.md`](SECURITY.md).

## Pull Requests

Small documentation fixes and clearly scoped bug fixes are welcome. Before
submitting larger changes, open an issue first so the intended direction can be
discussed.

By submitting a pull request, you confirm that you have the right to contribute
the code or documentation and that Inlay Cloud may use, modify, and distribute
your contribution as part of this repository and related Inlay products.

## Development

Clone:

```sh
git clone https://github.com/inlay-cloud/inlay-juce-module.git
```

Build and test:

```sh
make test
```
